## 代理工具

### `ssh -D <端口>` 代理

#### 1. 准备一台 linux 机器做为代理机器

> 如: 在阿里云申请一台香港的ECS服务器， 公网IP为: 1.1.1.1

#### 2. 本机使用 `-D` 参数 `ssh` 连接到该机器

```
ssh username@1.1.1.1 -D 10080
```

#### 3. 本机 web 配置代理上网

```
SOCKS5  127.0.0.1  10080
```

#### 4. 本机 win11 设置系统代理

```
设置 - 网络和Internet - 代理 - 手动设置代理 - 设置
使用代理服务器: 开启
代理IP地址: socks=127.0.0.1
端口: 10080
```
