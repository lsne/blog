# Obsidian 插件设置

```
Git 如果 obsidian 打开的目录是一个 git 仓库, 这个插件会自动 push 和 pull
Tray 系统托盘
```

#### Git 插件配置

```
Split automatic commit and push:        开启
Vault commit interval (minutes):        10
Auto Backup after stopping file edits:  开启
Vault push interval (minutes):          10
Auto pull interval (minutes):           5

Sync Method:                            Merge
Pull updates on startup:                开启
Push on backup:                         关闭
Pull changes before push:               开启
Automatically refresh source control view on file changes: 开启
Source control view refresh interval:   7000

Show status bar:                        开启
Show stage/unstage button in file menu: 开启
Show branch status bar:                 开启

其他选项全部为关闭状态或默认值
```

#### Tray 插件配置

```
Launch on startup:   开启
Hide on launch:      开启
Run in background:   开启
Hide taskbar icon:   关闭
Create tray icon:    开启

其他选项全部为关闭状态或默认值
```
