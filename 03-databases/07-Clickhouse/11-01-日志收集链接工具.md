# 日志收集链接工具

#### 日志数据采集和标准规范的统一

```
OpenTelemetry  是一款数据收集中间件。我们可以使用它来生成，收集和导出监测数据（Metrics,Logs and traces）
    opentelemetry项目开始于2019年，旨在提供基于云环境的可观测性软件的标准化方案，提供与三方无关的监控服务体系。项目迄今为止已获得了Zipkin, Jaeger, skywalking, Prometheus等众多知名中间件的支持。
    go语言的sdk地址: https://github.com/open-telemetry/opentelemetry-go

gohangout   一个 go 语言编写的 kafka 与clickhouse 同步数据的工具 https://github.com/childe/gohangout
filebeat 是个go写的二进制程序
Flume  是个java编写的二进制程序, 直接修改配置文件后启动, 读文件,然后可以输出到终端, kafka等多个目的端
Flink  是个 java 框架, 需要编写java代码才能真正使用起来
```
